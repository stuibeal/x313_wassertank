# X313 Wassertank

## ESP32 driven water tank control:
* TL-136 sensor for sensing water level
* bluetooth use with android app (MIT appinventor2)
* pump output for Mosfet switch
* valve output for Mosfet switch
* WS2812B Led ring for water level

All new build-up on an ARI458 'cause the guys in the werkhalle didn't it right.

Use the ESP-IDF (Eclipse) to compile and run.
